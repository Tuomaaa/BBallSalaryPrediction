2025-2026 NBA Player Value Model

This is a tool that predicts a player's market value (cap_pct) using an XGBoost regressor trained signing data to capture market prices. Used 14 engineered features including advanced metrics, physical attributes and contract history. This model is designed to reflect the value of a player if they were to sign a new contract without any CBA rules. This model is still under development, is not perfect and tend to underestimate super stars.
